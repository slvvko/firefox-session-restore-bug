    About this page

    This page demonstrates a Firefox bug where session restore (Ctrl+Shift+T or browser restart) overwrites DOM changes made after the initial page load.

    Firefox restores tabs from an internal DOM snapshot captured at load time. Any mutations made by JavaScript after load — whether user-triggered or automatic — are silently reverted.
